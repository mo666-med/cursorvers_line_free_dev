# AIガバナンスに関するメモ

## 重要な知見

- 医療AIを導入する前に、必ず「AI導入ガバナンス委員会」を設置すべき。厚労省のガイドラインでも推奨されている #cv_line #ai_gov

- AI診断支援ツールの「最終判断は医師が行う」は建前ではなく、法的責任の所在を明確にするために必須 #cv_line #law

- クリニックにAIを導入する際は、ベンダーとの契約書に「瑕疵担保責任」と「データ取り扱い規程」を必ず明記する #cv_line #law #ai_gov

## 税務・キャリア関連

- 当直バイトで稼ぐより、AIガバナンス顧問として月1回のZoom相談を受ける方が、時給換算で3倍以上になることがある #cv_line #career

- コンサルティング収入は、開業届を出せば「事業所得」として申告できる。書籍代やセミナー費が経費になる #cv_line #tax

## 一般的な気づき

- 「AIに詳しい医師」というポジションは、今後5年で最も価値が上がるキャリアの一つ #cv_line
