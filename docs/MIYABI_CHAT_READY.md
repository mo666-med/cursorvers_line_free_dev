# Miyabiチャットスクリプト - 正常動作確認

## ✅ スクリプトの状態

スクリプトは正常に起動し、`Miyabi >`プロンプトが表示されています。これは対話型モードで動作している状態です。

## 📝 次のステップ

`Miyabi >`プロンプトにコマンドを入力してください。

### 推奨コマンド

1. **ヘルプを表示**
   ```
   help
   ```

2. **Issue一覧を表示**
   ```
   issues
   ```

3. **特定のIssueを処理**
   ```
   issue 3
   ```

4. **終了**
   ```
   exit
   ```

## 🎯 使用例

```
Miyabi > help
Miyabi > issues
Miyabi > issue 3
Miyabi > exit
```

## ✅ 確認ポイント

- ✅ スクリプトが起動している
- ✅ `Miyabi >`プロンプトが表示されている
- ✅ 環境変数`OPENAI_API_KEY`が設定済み
- ✅ コマンド入力が可能な状態

## 🔧 トラブルシューティング

**コマンドが入力できない場合:**
- ターミナルにフォーカスが当たっているか確認
- カーソルが`Miyabi >`の後に表示されているか確認

**スクリプトを終了する場合:**
- `exit`と入力
- または`Ctrl+C`で強制終了
