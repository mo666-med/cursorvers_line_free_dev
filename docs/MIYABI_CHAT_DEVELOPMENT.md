# Miyabiチャットで開発を進める方法

## 🚀 開発を始めるには

Miyabiチャットが起動したら、以下のような自然言語で指示を出せます：

### 📋 Issue一覧を確認
```
Miyabi > Issue一覧を表示して
```

### 🔧 特定のIssueを処理
```
Miyabi > Issue #3を処理して
Miyabi > Issue #1を実行して
```

### 📝 新しい機能を追加
```
Miyabi > LINEウェルカムメッセージの設定ファイルを作成して
Miyabi > ドキュメントを改善して
```

### 🎯 開発タスクの例

#### Issue #3: LINEウェルカムメッセージ設定
```
Miyabi > Issue #3を処理して
```
- ウェルカムメッセージの設定ファイルを作成
- READMEに追記
- PRを作成

#### Issue #1: プロジェクト全体の推敲と改善
```
Miyabi > Issue #1を処理して
```
- READMEの改善
- コード品質の向上
- ドキュメントの整備

#### Issue #2: Manus API連動の実装
```
Miyabi > Issue #2を処理して
```
- Manus API連動の実装
- LINE EventとManus APIの統合

## 💡 ヒント

- 自然言語で指示を出すだけで、Miyabiが適切なアクションを実行します
- `help`コマンドで利用可能なコマンドを確認できます
- `exit`または`quit`でチャットを終了します
